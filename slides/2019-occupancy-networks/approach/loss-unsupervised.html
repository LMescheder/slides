## Loss Function

Loss for generative model (VAE):

$$\mathcal L^{\text{gen}}_{\mathcal B}(\theta, \color{blue}{\psi}) = \frac{1}{|\mathcal B|} \sum_{i=1}^{|\mathcal B|} \Bigl[\sum_{j=1}^ K \mathcal L(f_\theta(s_{ij}, \color{olive}{z_i}), o_{ij}) \\ + \color{red}{\mathrm{KL}} \left(\color{blue}{q_\color{blue}{\psi} (z \mid \color{black}{(s_{ij}, o_{ij})_{j=1:K}})} \mid \color{orange}{p_0(z)} \right) \Bigr] $$
Symbol Meaning
$\color{blue}{q_\psi(\cdot\mid\cdot)}$ Gaussian inference model
$\color{darkorange}{\mathcal{p_0}}$ Gaussian prior on latent variable
$\color{olive}{\mathcal{z_i}}$ Random sample from $\color{blue}{q(\cdot\mid\cdot)}$
$\color{red}{\mathrm{KL}}$ Kullback-Leibler divergence
