## Loss Function
Loss for supervised model:
$$ \mathcal L_\color{darkorange}{\mathcal{B}} (\color{blue}{\theta}) = \frac{1}{|\color{darkorange}{\mathcal{B}}|} \sum_{i=1}^{|\color{darkorange}{\mathcal{B}}|} \sum_{j=1}^ K \color{red}{\mathcal{L}} ( \color{darkgreen}{f}_\color{blue}{\theta} (\color{purple}{p_{ij}}, \color{brown}{x_i}), \color{olive}{o_{ij}} ) $$
Symbol Meaning
$\color{darkgreen}{f}$ Occupancy Network
$\color{blue}{\theta}$ Parameter vector
$\color{darkorange}{\mathcal{B}}$ batch of data samples $( \color{brown}{x_i}, \color{purple}{p_{ij}}, \color{olive}{o_{ij}} )$
$\color{brown}{x_i}$ $|\mathcal{B}|$ observations (e.g. images)
$\color{purple}{p_{ij}}$ $|\mathcal{B}| \times K$ random locations
$\color{olive}{o_{ij}}$ $|\mathcal{B}| \times K$ ground truth occupancies
$\color{red}{\mathcal L}$ cross-entropy loss
