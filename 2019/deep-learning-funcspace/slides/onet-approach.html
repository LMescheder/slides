--- ## Loss Function
Loss for supervised model:
$$ \mathcal L_{\mathcal B}(\theta) = \frac{1}{|\mathcal B|}\sum_{i=1}^{|\mathcal B|} \sum_{j=1}^ K \mathcal L(f_\theta(p_{ij},x_i), o_{ij}) $$
Loss for unsupervised model:
$$\mathcal L^{\text{gen}}_{\mathcal B}(\theta, \color{red}{\psi}) = \frac{1}{|\mathcal B|} \sum_{i=1}^{|\mathcal B|} \Bigl[\sum_{j=1}^ K \mathcal L(f_\theta(p_{ij}, \color{red}{z_i}), o_{ij}) \\ + \color{red}{\mathrm{KL} \left(q_\psi(z | (p_{ij}, o_{ij})_{j=1:K}) \,\|\, p_0(z) \right) } \Bigr] $$
